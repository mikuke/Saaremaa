# Saaremaa üritused

Projekti tulemus: lihtne rakendus, mis kuvab üritusi saaremaasuvi.ee lehelt ka siis kui ühendust pole. See ei ole veebilehe raam vaid
pigem sisustab native rakendust veebilehelt pärit andmetega. 
Kui aega jääb, siis muudab ka stiili sarnaselt veebilehele.
Kuidas: Võtan html-koodist andmed ning tekitan nendest Androidi rakenduse sisu.
Millega: Android Studio
